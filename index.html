O que é Git

Git é um software de controle de versão gratuito e de código aberto, que permite acompanhar alterações em arquivos. Ele é muito usado para desenvolvimento de software, tanto comercial quanto de código aberto.

O que é Github

O GitHub é uma plataforma online que permite armazenar, compartilhar e colaborar em projetos de desenvolvimento de software. É uma espécie de rede social para desenvolvedores.
