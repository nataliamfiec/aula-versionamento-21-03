O que é Git

O Mustang Shelby é uma versão de alto desempenho do Ford Mustang. A Shelby American construiu o modelo de 1965 a 1967, e a Ford Motor Company de 1968 a 1970. Em 2005, a Ford reviveu a marca Shelby para a quinta geração do Mustang.

Características:

º O Mustang Shelby GT500 é um cupê de duas portas, capaz de transportar até quatro passageiros

º O motor dianteiro abriga um DOHC V8 de 32 válvulas superalimentado com um supercharger e intercooler

º O Shelby GT500 tem uma velocidade máxima de 321 km/h

º O Shelby Super Snake 2024 é a primeira incursão da Shelby sobre a plataforma da nova geração do Mustang, a S650

O que é Github

O GitHub é uma plataforma online que permite armazenar, compartilhar e colaborar em projetos de desenvolvimento de software. É uma espécie de rede social para desenvolvedores.
